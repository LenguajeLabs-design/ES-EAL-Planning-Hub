# ES EAL Planning Hub

SSIS Elementary Language Support planning site for Grades 1-5.

## Contents

- `index.html` - main site
- `styles.css` - site styles
- `app.js` - simple interactions
- `docs/` - source reference documents

## Purpose

This site brings together SSIS elementary language support guidance, WIDA-based planning, and teacher reference documents in one place.
